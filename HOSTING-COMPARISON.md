## ⚡ Setup Rápido

### Azure (5 min)
```
1. Criar conta Azure (grátis)
2. Static Web Apps → Create
3. Conectar GitHub
4. Deploy automático!
```

### GitHub Pages (3 min)
```
1. Ativar Pages nas Settings
2. Selecionar GitHub Actions
3. Push para main
4. Deploy automático!
```

### Vercel (2 min)
```
1. Fazer login
2. New Project → Import
3. Selecionar repo
4. Deploy!
```
