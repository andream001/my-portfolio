## 🚀 Deploy Rápido em 3 Passos

### Opção 1: Azure (Recomendado) ⭐

```bash
# 1. Commit
git add .
git commit -m "feat: ready for azure"
git push origin main

# 2. No Azure Portal, conecte seu GitHub
```

### Opção 2: GitHub Pages

```bash
# 1. Push
git push origin main

# 2. Deploy automático via GitHub Actions
```

### Opção 3: Vercel

```bash
# 1. Acesse vercel.com
# 2. Clique: Import
# 3. Selecione seu repositório
# 4. Deploy!
```
